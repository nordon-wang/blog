## 使用技术

- 服务器渲染页面
- express 框架使用的中间件
    - blueimp-md5       MD5加密
    - body-parser       普通的请求处理
    - multer            文件上传请求处理
    - consolidate       处理模板引擎
    - cookie-parser     cookie处理
    - cookie-session    session处理
    - express-router    路由处理
    - express-static    静态文件处理
    - lodash            工具函数
    - moment            日期处理
    - mysql             mysql处理
